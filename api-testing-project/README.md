# ⚙️ API Testing Project (Advanced)

Este repositorio está destinado a un framework de pruebas de API más robusto, enfocado en la validación de reglas de negocio y seguridad.

## 🎯 Objetivos del Proyecto
- **Validación de Esquemas:** Implementación de JSON Schema Validation para asegurar contratos de datos.
- **Data-Driven Testing:** Uso de variables dinámicas y archivos de datos externos.
- **Integración con IA:** Utilización de herramientas como Claude Code y Cursor para la generación de casos de borde (edge cases) y optimización de scripts.
- **Seguridad:** Pruebas de autenticación y autorización (Bearer Tokens, API Keys).

## 🛠️ Stack Tecnológico Proyectado
- Postman / Newman
- JavaScript (Chai Assertion Library)
- GitHub Actions (para CI/CD)
